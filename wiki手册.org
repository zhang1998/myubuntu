### 3.1.2 令gollum支持中文链接

安装gollum-rugged_adapter, 运行gollum时, 加上参数 --adapter 为 rugged
```
sudo apt-get -y install cmake
sudo gem install gollum-rugged_adapter

gollum --port 4567 --adapter rugged 
```
